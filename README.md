# Bike Buyers Data Analysis (Excel Dashboard)

# Overview
This project analyzes customer demographic and purchasing data to identify factors influencing bike purchase decisions. An interactive Excel dashboard was developed to visualize trends and support data-driven insights.

# Objective
To examine customer attributes such as income, age, occupation, and marital status to determine patterns associated with bike purchases.

# Tools & Techniques
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Descriptive Analysis

# Dataset Description
The dataset contains customer-level information including:
- Age
- Gender
- Income
- Occupation
- Marital Status
- Purchase Decision (Yes/No)

# Key Insights
- Customers in the mid-income range showed higher purchase probability.
- Married individuals demonstrated stronger purchase tendencies.
- Age group trends indicated higher buying behavior among working-age customers.

# Dashboard Features
- Interactive filters using slicers
- Dynamic pivot-based summaries
- Visual comparison of buyers vs non-buyers
- KPI-focused summaries

# How to Use
1. Open `bike_buyers_dashboard.xlsx`
2. Use slicers to filter demographic groups
3. Review pivot charts for pattern analysis

# Business Value
The dashboard enables segmentation analysis and supports targeted marketing strategies for higher conversion rates.
